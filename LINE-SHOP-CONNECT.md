# คู่มือ "เชื่อม LINE ร้าน" — ส่งให้ร้านค้าทำตาม

ข้อความสำเร็จรูปด้านล่าง copy ส่งให้ร้านทาง LINE ได้เลย
พอร้านส่งรหัส (C… หรือ U…) กลับมา → วางใน Studio → ร้านค้า → ช่อง
"LINE ของร้านที่รับคำขอจอง/สั่งซื้อ" → เลือกโหมดส่งต่อ → Publish

---

🔗 วิธีเชื่อม LINE ร้านกับระบบคิวจอง aquamx (ทำครั้งเดียว ~2 นาที)

**แบบกลุ่ม (แนะนำ — พนักงานเห็นคิวพร้อมกันทุกคน)**
1. ตั้งกลุ่มไลน์ใหม่ เช่น "คิวจอง – ชื่อร้าน" ใส่เจ้าของ + พนักงาน (บัญชีไลน์ส่วนตัว)
   ⚠️ ห้ามมี Official Account อื่นอยู่ในกลุ่ม — LINE ให้มี OA ได้กลุ่มละ 1 ตัวเท่านั้น
2. ให้คนใดคนหนึ่งเพิ่มเพื่อน @aquamx ก่อน: https://line.me/R/ti/p/@aquamx
3. เชิญ "aquamx" เข้ากลุ่ม แล้วพิมพ์คำว่า `id`
4. bot จะตอบ `📋 LINE Group ID: C…` → ส่งรหัสนั้นกลับมาให้เรา
   (พิมพ์ `id` ถามซ้ำได้ทุกเมื่อ)

**แบบส่วนตัว (เจ้าของรับคิวคนเดียว)**
1. เพิ่มเพื่อน @aquamx: https://line.me/R/ti/p/@aquamx
2. ทักแชทหา aquamx ด้วยคำว่า `id`
3. bot จะตอบ `📋 LINE User ID: U…` → ส่งรหัสนั้นกลับมาให้เรา
   (พิมพ์ `id` ถามซ้ำได้ทุกเมื่อ)

หมายเหตุ: ใช้ LINE Official Account ของร้านรับคิวไม่ได้ (ระบบ LINE ไม่อนุญาตให้
OA ส่งข้อความหากันเอง) ต้องเป็นกลุ่มหรือบัญชีส่วนตัวเท่านั้น

## กติกา LINE ที่เกี่ยวข้อง (อ้างอิง)

- **1 กลุ่มมี OA ได้ 1 ตัว** — "At any time, only one LINE Official Account can be
  in a group chat or multi-person chat."
  https://developers.line.biz/en/docs/messaging-api/group-chats/
  ตัวที่สองที่ถูกเชิญจะถูกเด้งออกทันที (อาการเหมือนเชิญไม่สำเร็จ)
- **OA ส่งหา OA ไม่ได้** — push API ส่งได้เฉพาะ userId (U…) / groupId (C…)
  ของบัญชีคนจริงเท่านั้น
- **ส่งหารายคนได้เฉพาะคนที่เป็นเพื่อนกับ @aquamx** — ถ้าบล็อค/ลบเพื่อน ข้อความไปไม่ถึง
- ร้านที่สมัครผ่านเว็บด้วย LINE login: ระบบเติมช่องนี้ให้อัตโนมัติอยู่แล้ว
  (หน้า login ชวนแอดเพื่อน @aquamx ในตัว) — คู่มือนี้ใช้กับร้านที่เราสร้างให้ใน Studio
